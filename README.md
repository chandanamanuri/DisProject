# 📚 Book Club Web App

A full-stack ASP.NET MVC application designed to manage and engage a virtual book club community. Built using C#, SQL Server, and JavaScript, this platform allows users to explore featured books, register as members, and gain insights through interactive visualizations.

## 🌐 Live Demo
🔗 [Visit the Website](https://disproject20240421210910.azurewebsites.net)

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript, Razor Views
- **Backend**: ASP.NET MVC, C#
- **Database**: SQL Server
- **Hosting**: Azure Web Apps

## ✅ Key Features
- **Home Page**: Highlights book club news and upcoming events.
- **About Us**: Showcases the mission, team, and background of the club.
- **Featured Books**: Displays top book recommendations with reviews and cover images.
- **Membership Page**: Lists benefits and encourages users to join.
- **Book Insights**: Visualizes popular book genres, formats, and languages using dynamic charts.
- **User Sign-Up**: Allows new members to register securely.

## 🚀 Getting Started

### Open the Project in Visual Studio
- Open `DisProject.sln`

### Configure the Database
- Update your SQL Server connection string in `appsettings.json`

### Run the Application
- Use IIS Express or Kestrel to launch the app locally.
- Navigate to `https://localhost:{port}`
